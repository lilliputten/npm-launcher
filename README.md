# npm-package-user-scripts-gui

GUI interface to `npm-package-user-scripts-list` command.

## Installation

For current project:

```shell
npm i -D npm-package-user-scripts-gui
```
Global:
```shell
npm i -g npm-package-user-scripts-gui
```
